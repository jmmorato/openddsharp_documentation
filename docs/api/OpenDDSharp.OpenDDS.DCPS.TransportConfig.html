SwapBytes

Gets or sets a value indicating whether a value of false causes DDS to serialize data in the source machine's native endianness; a value of true causes DDS to serialize data in the opposite endianness. The receiving side will adjust the data for its endianness so there is no need to match this option between machines. The purpose of this option is to allow the developer to decide which side will make the endian adjustment, if necessary. The default value is false.
