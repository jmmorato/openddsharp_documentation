Enum ReturnCode

Represents the various return code values that DDS operations return.

Namespace: OpenDDSharp.DDS
Assembly: OpenDDSharp.Standard.dll
Syntax
public enum ReturnCode : int

Fields

Name Description
AlreadyDeleted

The object target of this operation has already been deleted.
BadParameter

Illegal parameter value.
Error

Generic, unspecified error
IllegalOperation

An operation was invoked on an inappropriate object or at an inappropriate time (as determined by QoS policies that control the behaviour of the object in question). There is no precondition that could be changed to make the operation succeed.
ImmutablePolicy

Application attempted to modify an immutable QoS policy.
InconsistentPolicy

Application specified a set of policies that are not consistent with each other.
NoData

Indicates a situation where the operation did not return any data.
NotEnabled

Operation invoked on an Entity that is not yet enabled.
Ok

Successful return.
OutOfResources

Service ran out of the resources needed to complete the operation.
PreconditionNotMet

A pre-condition for the operation was not met.
Timeout

The operation timed out.
Unsupported

Unsupported operation or QoS policy setting. Can only be returned by operations that are optional or operations that uses an optional <Entity>QoS as a parameter.
