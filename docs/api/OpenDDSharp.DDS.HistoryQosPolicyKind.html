Enum HistoryQosPolicyKind

This enumeration defines the valid kinds of the HistoryQosPolicy Kind.

Namespace: OpenDDSharp.DDS
Assembly: OpenDDSharp.Standard.dll
Syntax
public enum HistoryQosPolicyKind : int

Fields

Name Description
KeepAllHistoryQos

specifies that all possible samples for that instance should be kept. When "keep all" is specified and the number of unread samples is equal to the "resource limits" property of MaxSamplesPerInstance then any incoming samples are rejected.
KeepLastHistoryQos

Specifies that only the last depth values should be kept. When a data writer contains depth samples of a given instance, a write of new samples for that instance are queued for delivery and the oldest unsent samples are discarded. When a data reader contains depth samples of a given instance, any incoming samples for that instance are kept and the oldest samples are discarded.
